Quick start
===========

Per provare questo progetto:

* installa nodejs
* installa ionic
* crea la piattaforma browser
* esegui il server ionic nella root di questo repository

```
$ sudo apt-get install nodejs
$ sudo npm install -g ionic
$ ionic platform add browser
$ ionic serve
```
